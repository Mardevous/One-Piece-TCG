# TCG-One-Piece

L'application a pour objectif de pouvoir commander des cartes One Piece 

On peut ajouter une carte, la consulter, la modifier, la supprimer

Le projet est réalisé avec JS, NodeJS, Postman et MongoDB.
Du fait que ce soit mon premier projet, je découvre ces technologies.

Le plus grand challenge pour moi est de faire fonctionner l'application dans un premier temps.

Il faut donc JS, NodeJS, Postman et MongoDB pour que le projet puisse fonctionner dans un premier temps.

Il est tout à fait possible de reprendre le projet et l'améliorer dans le temps.
Cependant, bien penser à créer un fichier .env pour le faire fonctionner.
